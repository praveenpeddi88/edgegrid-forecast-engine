Every number on this page rests on these assumptions. FACT = true today by design or observation. HEURISTIC = a defensible choice that could move. OPEN = a known limitation we have not closed yet. v3 introduces the 76-day-horizon stress test (a23–a26). If you read a number you don't trust, find the assumption that explains it.
Disposable design prototype. The forecast curve is the real production v4 LightGBM model — one model per meter — pushed 76 days into the future from a context cutoff of 2026-02-04 23:30. Each forecast day is independently primed from the matching same-day-of-week 7 days back (no recursive cascade). The dashed actual line is held-out MDMS truth for Feb 5 → Feb 12 only; the rest of the horizon is pure forecast against an unknown future. Any value on this page maps to an assumption ID in the ledger.
